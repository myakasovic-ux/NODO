1. Ubicación del Mapa Interactivo

Para cambiar las coordenadas que muestra el mapa al hacer clic en el botón "Maps".

Archivo: script.js

Busca aproximadamente la línea 314 dentro de la lógica del mapa.

// Cambia los números dentro de user coordinates: [latitud, longitud]
mapInstance = L.map('map').setView([-33.044796, -71.578108], 16);
Tip: Puedes obtener estas coordenadas desde Google Maps haciendo clic derecho en un lugar y seleccionando los números que aparecen.

2. Parámetros de Animación (Partículas Warp)

Para modificar el comportamiento, cantidad, colores y patrones de la animación de fondo en la sección "Modelo".

Archivo: script.js

La configuración se encuentra al inicio de la función initWarpAnimation, aprox. línea 521.

// 1. CANTIDAD DE PARTÍCULAS
const starCount = 5000;

// 2. PATRÓN DE MOVIMIENTO
// 1: Radial | 2: Espiral | 3: Caos | 4: Pulsar | 5: Agujero Negro | 6: Nebulosa
const movementPattern = 5;

// 3. ESQUEMA DE COLOR
// 1: Starlight | 2: Dinámico
const colorScheme = 2;

// 4. VELOCIDAD GLOBAL
const globalSpeed = 0.5;

Simplemente cambia los números resaltados para alterar radicalmente el efecto visual.

3. Fotos del Carrusel de Proyecto

Para agregar, quitar o cambiar las imágenes que se muestran en la sección "Proyecto Actual".

Archivo: index.html

Ubica el contenedor .carousel-track alrededor de la línea 219.

<!-- GRUPO 1: Renders Exteriores -->
<div class="image-group group-exterior active">
  <img src="assets/images/A1.jpg" class="carousel-img active">
  <!-- Agrega más líneas <img> aquí para más fotos -->
</div>

<!-- GRUPO 2: Planos / Interiores -->
<div class="image-group group-interior">
  <img src="assets/images/interior.png" class="carousel-img active">
</div>
Asegúrate de que las nuevas imágenes estén guardadas en la carpeta assets/images/.

4. Valores por Defecto (Calculadora)

Para modificar los valores iniciales con los que carga la calculadora (Monto, Tasa, Años).

Archivo: index.html

Busca la sección de la calculadora alrededor de la línea 430.

<!-- Monto Propiedad (UF) -->
<input type="number" id="calc-monto" value="2800" ...>

<!-- Tasa Anual (%) -->
<input type="number" id="calc-tasa" value="4.2" ...>

<!-- Plazo (Años) -->
<input type="range" id="calc-anos" value="25" ...>
